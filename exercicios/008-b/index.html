Código Fonte e Pré Formatação

O comando code faz com que o conteúdo tenha uma formatação visualmente mais intuitiva get.element(daaaaaad). Uma observação é que o tag CODE não indenta código, fica tudo bagunçado. Para isso temos que colocar o código entre a tag PRE. Confira abaixo um exemplo:


        
            Algoritmo "PRATICA 1"
            //PREENCHER UMA MATRIZ 3X2 COM VALORES NO TECLADO
            
            Var
            
            m : vetor [1..3, 1..2] de inteiro
            l,c : inteiro
            
            
            Inicio
            
            escreval ("Vamos fazer uma matriz de 3 linhas e 2 colunas.")
            escreval ("Preencha conforme solicitado.")
            
            para l := 1 ate 3 faca
                 para c := 1 ate 2 faca
                     escreva ("Digite o número que vai na linha ", l, " e na coluna ", c, ": ")
                     leia (m[l,c])
                 fimpara
            
            fimpara
            
            para l := 1 ate 3 faca
                para c := 1 ate 2 faca
                    se (m[l,c] % 2 = 0)  entao
                       escreval ("PAR ",m[l,c])
                    senao
                       escreval ("IMPAR ", m[l,c])
                    fimse
                fimpara
            fimpara
            
            Fimalgoritmo
